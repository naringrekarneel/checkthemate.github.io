# ♚ Check the Mate
[![Live Demo](https://img.shields.io/badge/Live-Demo-green)](https://naringrekarneel.github.io/checkthemate.github.io/)

**Check the Mate** is a beautifully crafted, highly intuitive Chess Companion App meticulously designed to elevate the traditional over-the-board chess experience for players of all skill levels. By seamlessly merging the robust functionality of a professional tournament chess timer with an automated material advantage tracker, this application completely eliminates the need for clunky physical clocks and tedious mental math. Instead, it empowers you to focus entirely on your strategy, board vision, and gameplay.

Built with premium aesthetics at its core, the application features a stunning, dark-mode native glassmorphic interface that looks incredibly sharp and modern on any screen. The core innovation lies within its effortless piece-capture interface: whenever an opponent's piece is captured, a simple tap on the corresponding beautifully rendered chess icon instantly updates the dynamic central advantage bar. This visual indicator mathematically calculates the material imbalance in real-time, providing both players with immediate, frictionless visual feedback regarding who is currently leading the game in material value.

The application truly sets itself apart through its unparalleled native cross-device responsiveness and thoughtful layout engineering. On desktops and larger tablet screens, it presents a gorgeous side-by-side dashboard layout that is absolutely perfect for live streaming, tournament broadcasting, or analyzing games alongside a physical board. However, when opened on a smartphone, the application intelligently detects the device format and automatically transforms into a highly specialized face-to-face competitive mode. In this unique mode, the layout smartly divides the screen exactly in half, rotating Player 2's entire interface a full 180 degrees. This ingenious design allows two players to lay the phone completely flat on the table between them and tap their respective clocks comfortably, perfectly mimicking the tactile experience of a traditional physical chess clock.

Furthermore, "Check the Mate" handles complex chess scenarios effortlessly. Should you need to promote a pawn to a Queen or Knight, a simple long-press gesture on your player avatar instantly triggers a fluid, perfectly mapped radial gesture menu. This allows you to quickly add advanced pieces to your captured score without ever breaking your intense concentration.

Under the hood, the entire application is engineered strictly with pure, vanilla web technologies—HTML5, CSS3, and JavaScript—guaranteeing lightning-fast performance, zero external framework dependencies, and flawless execution across all modern mobile and desktop browsers. From the urgent, pulsing visual cues that alert you when a player is running critically low on time, to the flawlessly smooth CSS animations and automatic device theme syncing, "Check the Mate" fundamentally redefines what a digital chess companion can and should be.

## 🛠️ Installation

Simply clone this repository and open `index.html` in any modern web browser. No build steps or package installations are required!

```bash
git clone https://github.com/yourusername/check-the-mate.git
cd check-the-mate
# Open index.html in your browser
```
